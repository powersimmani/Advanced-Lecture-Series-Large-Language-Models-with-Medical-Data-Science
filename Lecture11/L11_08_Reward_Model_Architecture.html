Reward Model Architecture
Model Structure
Reward models predict scalar scores for model outputs, learned from expert preference data to guide policy optimization.
Architecture Components
Base Encoder: Transformer model (BERT, GPT, etc.)
Pooling Layer: Aggregate sequence representations
Reward Head: Linear layer outputting single scalar
Loss Function: Bradley-Terry or ranking loss
Regularization: Dropout, weight decay to prevent overfitting
Training Process
Input: Pairs of outputs with preference labels
Forward Pass: Compute reward scores for both outputs
Loss Calculation: Penalize incorrect rankings
Optimization: Adam optimizer with learning rate scheduling
Validation: Monitor accuracy on held-out preference data
